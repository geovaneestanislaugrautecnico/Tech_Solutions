# 🚀 Projeto Web Tech Solutions - Curso de Front-End

Este repositório contém o projeto prático desenvolvido durante as aulas da disciplina de **Programação HTML, CSS e JavaScript**. O objetivo é aplicar os conceitos fundamentais de desenvolvimento web, evoluindo de uma estrutura básica até uma interface moderna, responsiva e dinâmica.

---

## 🎨 Sobre o Projeto

O projeto consiste em um site institucional para uma empresa fictícia de tecnologia chamada **Tech Solutions**. Recentemente, a interface foi refatorada para um estilo **Dark Mode Minimalista**, seguindo as tendências de design atuais do mercado de tecnologia (UI/UX).

### ✨ Funcionalidades e Características:
- **Estruturação Semântica:** HTML5 puro e acessível.
- **Design Moderno:** Estilização CSS3 com paleta dark minimalista e cores de destaque em Ciano Elétrico.
- **Responsividade:** Layout adaptável para dispositivos móveis (Smartphones e Tablets).
- **Interatividade:** Validação de formulários e manipulação dinâmica do DOM com JavaScript.

---

## 🛠️ Tecnologias Utilizadas

*   **HTML5** - Estruturação de conteúdo.
*   **CSS3** - Estilização, Grid/Flexbox e transições modernas.
*   **JavaScript (ES6)** - Lógica de programação e comportamento do lado do cliente.
*   **Google Fonts Icons** - Pacote de ícones minimalistas em vetor.

---

## 📈 Histórico de Versionamento

O projeto foi dividido em marcos de aprendizado para refletir o conteúdo programático da matéria:

### 🔹 v1.0.0 — Estrutura Inicial (HTML5)
*   Criação das tags semânticas principais (`<header>`, `<nav>`, `<section>`, `<footer>`).
*   Construção do formulário de contato com diferentes tipos de `input`.

### 🔹 v2.0.0 — Estilização e Identidade Visual (CSS3)
*   Aplicação de reset de margens e fontes globais.
*   Alinhamento de elementos e criação de componentes de cards utilizando Flexbox.
*   Implementação do primeiro layout responsivo com `@media` queries.

### 🔹 v3.0.0 — Modernização e Tema Dark (Refatoração UI/UX)
*   Inversão da paleta de cores para um estilo *Dark Mode Minimalista*.
*   Inclusão de animações suaves (`transition`) com curvas de velocidade premium (`cubic-bezier`).
*   Substituição de imagens estáticas por ícones de fonte dinâmicos (Google Icons).

### 🔹 v4.0.0 — Interatividade e Comportamento (JavaScript)
*   Manipulação de eventos de clique (`addEventListener`).
*   Alteração dinâmica de conteúdo e estilos do DOM.
*   Validação lógica de campos obrigatórios no formulário de contato.

## 🚀 Como Executar o Projeto Localmente

1. Faça o clone deste repositório:
   ```bash
   git clone https://github.com
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd nome-do-repositorio
   ```
3. Abra o arquivo `index.html` diretamente em seu navegador ou utilize a extensão **Live Server** no VS Code para uma melhor experiência.
